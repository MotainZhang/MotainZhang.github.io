Tomcat的简单总结

发表于 2017-05-06 | 分类于 Java Web ， 服务器 |

Tomcat是什么

Tomcat是一个免费、开源的，用Java编写的Java Web服务器。

Tomcat的安装及配置

  1. 官网下载压缩包，解压到合适文件目录下。
  2. 配置环境变量CATALINA_HOME和CATALINA_BASE，两者内容相同
    • 需要Java配置相关环境变量：

运行bin目录下的startup.bat/shutdown.bat，即可启动/关闭 Tomcat。
浏览器通过localhost:8080即可访问Tomcat本地展示的首页。

部署项目到Tomcat

在Tomcat目录下，手动部署项目繁琐且效率低下。
可以通过Maven的Tomcat插件进行项目的开发调试。

可以通过首页右侧的Manager App管理web项目。
首次使用Manager App需要修改 %CATALINA_HOME%/conf/tomcat-users.xml文件
